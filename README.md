# 🎓 Student Marks Analysis System

## 📌 Description

This is a Python-based console application developed to manage and analyze student academic performance.
The program allows users to input student details, calculate averages, assign grades, and perform basic class-level analysis.

## 🚀 Features

* ➕ Add student details (name + 5 subject marks)
* 📋 Display all student records
* 🏆 Identify class topper
* 📊 Calculate class average
* 🎯 Automatic grade assignment (A–F)
* ✅ Pass/Fail status evaluation

## 🛠️ Technologies Used

* Python
* Lists & Dictionaries
* Conditional statements
* Loops & Functions
